# Scripts de développement — La RST

Ce dossier contient les scripts utilisés pour faciliter le travail en environnement local.

## Contenu
- **start-dev.sh** : lance le backend en mode développement.
- **reset-db.sh** : réinitialise la base PostgreSQL locale (sans données sensibles).

## Objectif
Fournir un environnement simple, reproductible et adapté aux contributeurs.
