# Politique de données — La RST

La RST traite des données personnelles sensibles (bénévoles, bénéficiaires).  
La plateforme adopte une approche éthique et conforme au RGPD.

---

## 1. Principes

- **Minimisation** : seules les données nécessaires sont collectées.
- **Transparence** : les utilisateurs savent quelles données sont stockées.
- **Sécurité** : accès restreint, stockage protégé.
- **Contrôle** : possibilité de modifier ou supprimer les données.

---

## 2. Données collectées

### Bénévoles
- Identité
- Contact
- Compétences
- Disponibilités
- Notes internes

### Bénéficiaires
- Identité
- Contact
- Type d’accompagnement
- Référent bénévole
- Notes internes

---

## 3. Stockage

- Base PostgreSQL
- Accès limité aux administrateurs
- Sauvegardes locales ou chiffrées

---

## 4. Droits des personnes

- Accès
- Rectification
- Suppression
- Opposition

---

## 5. Évolutions prévues

- Export des données
- Journalisation des accès
- Politique de conservation
